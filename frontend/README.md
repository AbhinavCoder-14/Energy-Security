# Aegis Energy — Frontend

Next.js (App Router) command-center dashboard bound to `UnifiedDashboardPayload`.

## Design
Vercel monochrome Geist system (`DESIGN-vercel.md` / `skills.md`): canvas `#fafafa`, ink `#171717`, hairline cards, pill scenario CTAs.

## Run
```bash
npm install
npm run dev
```

Requires backend at `http://localhost:8000` (override with `NEXT_PUBLIC_API_BASE`).
